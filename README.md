#  Study Planner : Low-Fidelity Wireframe

##  Problem Statement

Students often struggle to keep track of assignments, deadlines, and study tasks. This usually leads to missed submissions and poor time management. The goal of this project is to design a simple mobile application that helps students organize their tasks and stay on top of their work without feeling overwhelmed.

---

##  User Research

Basic informal research was conducted by discussing with a few students. It was observed that most students:

* Forget deadlines when handling multiple subjects
* Feel overwhelmed when using complex productivity apps
* Prefer simple, quick, and easy-to-use interfaces

This insight led to focusing on **simplicity and clarity** in the design.

---

##  User Personas

###  Aarav :– The Struggling Student

* **Age:** 20
* **Background:** College student managing multiple assignments
* **Goals:** Keep track of tasks and avoid missing deadlines
* **Pain Points:**

  * Often forgets deadlines
  * Gets overwhelmed with too many tasks
* **Needs:** A simple and distraction-free interface

---

###  Priya :– The Organized Planner

* **Age:** 21
* **Background:** Well-organized student who plans ahead
* **Goals:** Maintain a structured schedule and track progress
* **Pain Points:**

  * Needs a clear overview of tasks
  * Prefers structured and well-arranged layouts
* **Needs:** Clean and organized task management system

---

## User Flow

The application follows a simple and logical flow:

Login → Dashboard  

Dashboard → Task Creation 

Dashboard → Focus Timer  

Dashboard → Calendar

This flow ensures that users can quickly add and view tasks without unnecessary steps.

---

##  Wireframes

Low-fidelity wireframes were designed to represent the structure of the application.


- **Login Screen:** Allows users to securely access the application using their credentials. It keeps the entry process simple and quick to avoid unnecessary friction.

- **Dashboard Screen:** Acts as the main hub where users can view their tasks and deadlines at a glance. It provides a clear and organized overview for efficient task management.

- **Task Creation Screen:** Enables users to add new tasks with essential details such as task name, deadline, and notes. The design is kept minimal to make task entry fast and easy.

- **Calendar View:** Helps users visualize upcoming deadlines and plan tasks more effectively.
  
- **Focus Timer:** Allows users to concentrate on study sessions using timed intervals, improving productivity.

---

## Design Thinking Process

The design process started by identifying a common problem among students — difficulty in managing tasks and remembering deadlines. Based on basic user research, it was clear that users prefer simple and easy-to-use applications rather than complex systems.

Two types of users were considered: one who struggles with organization and another who prefers structured planning. This helped in balancing simplicity and clarity in the design.

The application was designed with a minimal interface to avoid overwhelming users. The dashboard provides a clear overview of tasks and deadlines, making it easy to understand at a glance.

Additional features like the Calendar and Focus Timer were included to improve usability. The Calendar helps users visualize upcoming deadlines, while the Focus Timer supports better concentration during study sessions.

Overall, the design focuses on creating a straightforward and efficient user experience, ensuring that users can manage tasks quickly without confusion.
